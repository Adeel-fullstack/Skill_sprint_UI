# 🎨 Skill Sprint UI

A modern and responsive **front-end UI project** built using **Vue.js and Tailwind CSS**.  
This project is designed for skill learning platforms, dashboards, or educational web applications with a clean and user-friendly interface.

---

## 🚀 Features
- ⚡ Modern and responsive UI design  
- 🧩 Reusable Vue components  
- 🎯 Clean and structured navigation  
- 🎨 Styled with Tailwind CSS utility classes  
- 🖥️ Optimized for learning platforms and dashboards  

---

## 🛠️ Tech Stack
- Vue.js  
- Tailwind CSS  
- HTML5  
- JavaScript (ES6+)

## ▶️ How to Run the Project Locally

Follow these steps to run the project on your system:

### 1. Clone the repository
```bash
git clone https://github.com/your-username/skill-sprint-ui.git
2. Navigate to the project folder
cd skill-sprint-ui
3. Install dependencies
npm install
4. Run the development server
npm run dev
🌐 Build for Production

To create a production-ready build:

npm run build
